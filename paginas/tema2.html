¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar que se utiliza para crear el contenido de las páginas web. Define la estructura de la información usando etiquetas.

Estructura básica de una página HTML

Una página HTML básica suele tener esta estructura:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
</head>
<body>
    Contenido visible de la página
</body>
</html>

Actividad sugerida

  1. Crea un archivo llamado prueba.html.
  2. Copia la estructura básica de HTML.
  3. Cambia el título y agrega un encabezado <h1> con tu nombre.
  4. Abre el archivo en tu navegador y observa el resultado.
